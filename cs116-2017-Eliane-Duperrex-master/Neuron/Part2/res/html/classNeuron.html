bool Neuron::update ( const step &  t,
const int &  poisson 
)

Update of the neuron for a step t.

test if neuron spike (if membrane potential > a threshold)
if spike -> addSpikeTime of t, return true
use of isRefractory
if neuron in refractory time -> potential set to 0 and return false
else -> potential updated -> with the differential equation
-> with the spike received from connexion (buffer)
-> with background noise (spike with poisson)
-> reset buffer to 0
-> return false
